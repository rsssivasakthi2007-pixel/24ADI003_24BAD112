Scenario 1: E-commerce Sales Data

Dataset Link : https://www.kaggle.com/datasets/carrie1/ecommerce-data

In this scenario, an e-commerce sales dataset was analyzed to understand product categories, sales behavior, and missing information. The dataset was loaded using Pandas and inspected using basic functions such as head(), info(), and describe(). Missing values were identified, and sales per product were visualized using bar and line charts to observe trends and top-performing products.

Scenario 2: Hospital Patient Records

Dataset Link :https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database

In this scenario, a hospital patient dataset was explored to analyze health metrics and identify missing values. The structure of the dataset was examined, and key attributes such as glucose level and age were visualized using histograms and boxplots. Patterns related to diabetes indicators and outliers in health measurements were observed.

Scenario 3: Housing Dataset

Dataset link :https://www.kaggle.com/datasets/harishkumardatalab/housing-price-prediction

In this scenario, a housing dataset was investigated to detect missing features and understand relationships between house attributes and prices. The dataset was inspected, missing values were summarized, and correlations between numerical features were visualized using scatter plots and a heatmap. These visualizations helped identify important factors influencing house prices.

Scenario 4: Customer Personality Analysis

Dataset Link :https://www.kaggle.com/datasets/imakash3011/customer-personality-analysis

In this scenario, customer demographic and spending data were analyzed to identify missing account information and understand customer behavior. The dataset structure and null values were examined, and visualizations such as bar charts and box plots were used to study age, income, and spending patterns. The analysis provided insights into customer segmentation and purchasing trends.
